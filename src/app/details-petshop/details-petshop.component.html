PETSHOP

Détails du plateforme

PETSHOP est un projet de site web e-commerce dédié à la vente de produits pour animaux de compagnie. Ce projet vise à offrir une plateforme intuitive et sécurisée où les utilisateurs peuvent explorer, sélectionner, et acheter des articles pour leurs animaux domestiques.

Conception et Architecture: La structure du site a été soigneusement élaborée à l'aide d'UML pour définir l'architecture du système et les différents processus commerciaux, assurant ainsi une base solide pour le développement.

Outil de conception des diagrammes : LUCIDSPARK est utilisé pour tracer les diagrammes.

Outils de conception UI/UX : Une interface utilisateur attrayante et facile à utiliser a été conçue avec Figma pour offrir une expérience d'achat fluide et agréable, mettant en avant la commodité et l'accessibilité.

Technologies frontend : Le front-end du site a été développé en utilisant Angular, permettant une navigation rapide et une interface réactive qui s'adapte parfaitement aux besoins des utilisateurs.

Éditeur principal : VISUAL STUDIO CODE est l'éditeur principal.

Backend : Le back-end a été réalisé avec Java Spring Boot pour gérer les fonctionnalités serveur, tandis que MySQL a été utilisé pour la gestion des données, assurant une expérience utilisateur robuste, fiable, et sécurisée.

Test des API : INSOMNIA est l'outil utilisé pour tester les API backend.
