Détails du plateforme

La plateforme propose une gamme de fonctionnalités conçues pour répondre aux besoins spécifiques des consultants et des clients. Pour les clients, l'application offre une interface conviviale pour parcourir et sélectionner des consultations proposées par des experts dans le domaine spécifique. Ils peuvent facilement choisir une date de consultation parmi celles suggérées par le consultant, et effectuer le paiement en toute sécurité. Une fois la consultation confirmée, un lien de vidéoconférence est automatiquement généré pour faciliter la communication à distance. Pour les consultants, l'application simplifie la gestion de leurs consultations en ligne en leur permettant de définir leurs services, tarifs et disponibilités. Ils peuvent recevoir des demandes de rendez-vous des clients, valider les rendez-vous et suivre leur emploi du temps grâce à un tableau de bord intuitif. La génération automatique de liens de vidéoconférence et l'intégration de fonctionnalités de paiement sécurisées contribuent à optimiser le processus de consultation en ligne.

Approche intégrée : Une approche intégrée combine les méthodologies UML et Merise pour structurer les fonctionnalités et les données.

Outil de conception des diagrammes : LUCIDSPARK est utilisé pour tracer les diagrammes.

Technologies frontend : ANGULARJS et TYPESCRIPT sont choisis pour des interfaces dynamiques, avec CSS, HTML, JAVASCRIPT, BOOTSTRAP et MATERIAL UI pour le style et la structure.

Éditeur principal : VISUAL STUDIO CODE est l'éditeur principal.

Outils de conception UI/UX : FIGMA, WEBFLOW, CANVA et PHOTOSHOP sont utilisés pour la conception UI/UX.

Gestion des données : MYSQL et FIREBASE REALTIME DATABASE sont employés pour la gestion des données.

Bibliothèques et APIs : Diverses bibliothèques et APIs telles que STRIPE, ZEGOCLOUD et EMAILJS enrichissent les fonctionnalités.

Backend : JAVA SPRING BOOT est utilisé pour le backend, avec une architecture MICROSERVICES gérée via SPRING TOOL SUITE 4 et XAMPP pour les serveurs locaux.

Test des API : INSOMNIA est l'outil utilisé pour tester les API backend.

Gestion du projet : Une approche agile basée sur SCRUM est adoptée pour la gestion du projet, avec TRELLO pour l'organisation des tâches.
